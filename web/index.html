提示
交互
  • 点击地图上的点可查看该点的状态和时间
  • 时间标签常显：右下角图层面板现在支持按图层开启“标签常显”，勾选后无需点击，直接在地图上展示该图层的时间/停留信息
  • 时间轴增强：滚轮可低敏缩放时间范围；右上 < / > 或键盘 ← / → 可逐点移动光标；全时间条支持拖动窗口、拖拽边缘改范围；在时间轴上右键可给当前时间点打单点标签或启动段落标记
  • 日期窗口快标：底部日期条右侧可设置“跨度 x 天”；0 表示不限，大于 0 时起止日期会按固定天数联动。选好“整段标签”后可直接把当前日期窗口保存成整段标记，再次点击可取消或更新标签
  • 三列审阅箱：左侧按“待审 / 已通过 / 未通过或跳过”分栏展示 UID，点击任意卡片即可切换轨迹并加载 latest review
  • 筛选：搜索框同时匹配 UID、当前标注者 notes 和状态词；状态筛选支持 road/subway/railway/unmatch/stay 的任一命中或全部命中
  • 编辑与流转：已审 UID 再次打开后可直接修改并保存；保存后默认跳到当前筛选结果中的下一条待审 UID
  • 图层/样式：地图右下角提供折叠式浮动面板
图层含义（raw / snap / od / fmm / line）
raw：原始信令轨迹点（未处理）
snap：卡尔曼平滑后的轨迹点
od：OD 分割结果（驻留 stay / 移动 move 的起止点）
fmm：FMM 匹配后的点（投影到道路/地铁/铁路）
line：FMM 匹配后的路径线（按 segment 分段）
状态含义（fmm/line 点）
road：匹配到道路网
subway：匹配到地铁网
railway：匹配到铁路网
unmatch：未匹配到任何网络
stay：驻留点（OD 判定为停留）

标注设置
自定义段落类别、颜色、顺序，以及时间轴聚焦/未聚焦透明度。

段落类别

这些类别会直接出现在时间轴右键菜单里，用来发起“开始段落标记”。

整轨 Tag

这些 tag 会出现在顶部整轨标注条的下拉选单里，作为当前用户的常用标签池。

标记快捷键

在非输入状态下直接触发保留 / 排除 / 跳过 / 保存；点击按键框后按下新键即可覆盖，Backspace / Delete 可清空。
