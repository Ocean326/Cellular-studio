交互
  • 点击地图上的点可查看该点的状态和时间
  • 时间标签常显：右下角图层面板现在支持按图层开启“标签常显”，勾选后无需点击，直接在地图上展示该图层的时间/停留信息
  • 时间轴增强：滚轮可低敏缩放时间范围；全时间条支持拖动窗口、拖拽边缘改范围；在时间轴上右键可给当前时间点打单点标签或启动段落标记
  • 三列审阅箱：左侧按“待审 / 已通过 / 未通过或跳过”分栏展示 UID，点击任意卡片即可切换轨迹并加载 latest review
  • 筛选：搜索框同时匹配 UID、latest reviewer、latest notes；状态筛选支持 road/subway/railway/unmatch/stay 的任一命中或全部命中
  • 编辑与流转：已审 UID 再次打开后可直接修改并保存；保存后默认跳到当前筛选结果中的下一条待审 UID
  • 图层/样式：地图右下角提供折叠式浮动面板
图层含义（raw / snap / od / fmm / line）
raw：原始信令轨迹点（未处理）
snap：卡尔曼平滑后的轨迹点
od：OD 分割结果（驻留 stay / 移动 move 的起止点）
fmm：FMM 匹配后的点（投影到道路/地铁/铁路）
line：FMM 匹配后的路径线（按 segment 分段）
状态含义（fmm/line 点）
road：匹配到道路网
subway：匹配到地铁网
railway：匹配到铁路网
unmatch：未匹配到任何网络
stay：驻留点（OD 判定为停留）
